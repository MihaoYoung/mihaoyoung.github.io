Linux多节点环境打造

发表于 2021-02-21 分类于 Linux 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

基本准备

软件资源准备

Linux CentOS7

进入CentOS官网，点击CentOS Linux

选择CentOS7下的x86_64

选择合适镜像链接下载CentOS-7-x86_64-DVD-2009.iso

也可以点击安装链接直接安装

VMware Workstation Pro（虚拟机软件）

此软件适用于window和Linux操作系统

点击下载链接选择合适的版本下载

若使用mac操作系统，请下载VMware fusion

Xshell（SSH终端）

在该网站提交邮箱即可获得免费版下载地址

Xftp（SFTP文件传输工具）

在该网站提交邮箱即可获得免费版下载地址

安装Linux

打开VMware，点击创建新的虚拟机

选择-典型

选择-稍后安装操作系统

选择-CentOS7 64位版本

设置名称和路径

分配磁盘大小，实际占用的是刚刚我们选择路径的磁盘空间

自定义硬件

选择刚刚下载CentOS7 ios文件

设置处理器内核数量，博主笔记本为8核处理器，因此设置2核

设置内存，博主笔记本为8g内存（打算以后再升级），设置2g内存

点击完成，然后启动

按下键盘↑方向键，回车

然后选择语言（选中文），点击继续，选择自己安装的软件，点击安装

设置root密码

静静等待安装~

最后创建用户名即可
