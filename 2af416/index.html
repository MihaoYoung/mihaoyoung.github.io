Java 类的初始化顺序

静态变量，静态代码块>主函数main>变量，构造代码块>构造方法

构造函数

  • 只有创建对象，构造函数才会执行
  • 只运行一次
  • 给对象进行初始化

构造代码块

  • 创建对象时执行，且优先于构造函数
  • 无法自行调用
  • 给对象经行统一初始化

静态代码块

  • 随着类的加载而执行，且只执行一次，优先于主函数
  • 给类进行初始化
  • 块中变量为局部变量
  • 一个类可以有多个静态代码块

类的加载时机

  • new 创建该类对象时
  • 使用java.lang.reflext进行反射调用时
  • 无事先加载该类，而加载该类的子类时
  • 虚拟机启动，初始化main函数所在的类

执行顺序

  • 执行父类静态代码块，初始化父类静态成员变量
  • 执行子类静态代码块，初始化子类静态成员变量
  • 执行父类构造代码块，父类构造函数，父类普通成员变量
  • 执行子类构造代码块，子类构造函数，子类普通成员变量
